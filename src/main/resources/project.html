Table of Contents

Roughly Enough Items is a mod alternative for JEI / NEI while they don't support 1.13 - 1.14, this project will be abandoned once better recipe viewing mods are updated.

 

1.13's version requires Rift.

1.13.2's version requires Chocohead's Rift Fork.

1.14's version requires Fabric Mod Loader and the Fabric API. (Optional: Plugin Loader for loading REI plugins)

 

NOTE: 1.5.1 is the last feature update for 1.13.0 as it is too hard for me to maintain 3 versions oh my

 

Currently Supported Languages:
- English

- Chinese Traditional

- Chinese Simplified

- French

- German (Soon™)

- Estonian (Soon™)

 

 

Changelog:

https://minecraft.curseforge.com/projects/roughly-enough-items/pages/changelog

 


 

Features Added from JEI

  • Better Searching
  • Accepts Searching by Tooltip (Prefix: #)
  • Accepts Excluding (Prefix: -)
  • Accepts Multiple Searching (Use: |)
Example: -@randommod stone will search items not from randommod named stone
  • Fixed Creative Gui
  • Potion Recipe (Experimental)
  • Center Search Box
  • Move Items to Crafting Area
  • Custom Ordering
  • More should be on https://shedaniel.me/RoughlyEnoughItems/

Features unique to REI

  • Show only craftables button

Features that I will work on in the future

  • View To-Do List here. And if they are not there, please suggest some <3

I know this is still buggy and basic but I am trying to make this better :)

Roughly Enough Items 是一個暫時替代 NEI / JEI 的模組，一旦有更好的模組可以替代，此模組會被取消。


1.13 需要 Rift

1.13.2 需要 Chocohead的Rift

1.14 需要 Fabric Loader 和 Fabric API (如要安裝REI插件，請安裝 Plugin Loader)

 

目前支持的語言：
- 英語

- 繁體中文

- 簡體中文

- 法文

- 德文（很快™）

- 愛沙尼亞語（很快™）


注意：1.5.1是1.13.0版的最後一個功能更新，同時維護3個版本實在是太累了

 

更新:
https://minecraft.curseforge.com/projects/roughly-enough-items/pages/changelog

 

從JEI中藉鑑的功能:

- 更好的搜索功能

- 修正了創造模式下的Gui

- 藥水配方功能

- 修訂了滾動

- 每頁現在顯示1/2個配方

- 搜索框居中

- 將物品移入合成框

- 自定義排序

- 更多信息參考: https://shedaniel.me/RoughlyEnoughItems/

未來會新增和改進的功能:
在這裡查看計劃中的功能列表，如果裡面沒有你的想法，可以進行提議。
